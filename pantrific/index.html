How It Works

From how you want to feel to a meal on your plate

1

Tell Us How You Want to Feel

During onboarding, share your wellbeing goals better sleep, more energy, improved mood along with any nutrient deficiencies or dietary needs that will be accounted for.

🎯
2

We See What You Have

A Raspberry Pi camera scans your fridge and pantry using YOLO object detection. It identifies every ingredient automatically so nothing goes to waste.

📷
3

AI Builds Your Recipes

Spoonacular AI and Gemini AI analyses the micro and macronutrients and combines your ingredients, your goals, and nutritional science to generate personalised meals. Each recipe is chosen because it moves the needle on how you feel.

🧠
4

Cook, Eat, Feel Better

Follow step-by-step instructions, log what you eat, and track how your nutrient intake trends against your wellbeing targets over time.

✨

What We Built

Every layer of Pantrific was designed and engineered by our team

🤖

Custom Vision Model

  • Built a custom dataset and trained a YOLOv26m model locally to detect pantry items
  • Deployed the model to both Raspberry Pi and server for flexible inference
  • Capable of running fully offline with zero cloud dependency
  • Built a Python service to structure and relay detections to the API
⚙️

Backend API

  • Designed a RESTful API in Fastify with TypeScript
  • Wrote a full relational schema with Drizzle ORM & Postgres
  • Implemented end-to-end request validation with Zod
  • Built custom token-based authentication from scratch
📱

Mobile App

  • Built a native Android app with Expo & React Native
  • Engineered optimistic updates and real-time data sync
  • Designed a custom warm UI system with NativeWind
  • Created animated interactions and parallax effects
🧠

AI Pipeline

  • Engineered structured prompts that map goals to nutrients
  • Built a dual-AI pipeline using Gemini and Spoonacular
  • Designed a two-level caching system for fast responses
  • Created Zod schemas to enforce reliable AI output
📦

Monorepo Architecture

  • Structured the codebase as a pnpm monorepo
  • Set up TypeScript project references for type safety
  • Authored shared schemas & utilities across all apps
  • Managed Python and Node dependencies side by side
🏗️

Infrastructure

  • Dockerised database
  • Wrote and managed migrations with Drizzle Kit
  • Implemented secure credential storage on-device
  • Configured environment-based deployment across services

AI + Coding

Training custom models and writing the code that powers Pantrific

Training YOLOv26m on a custom food dataset using Apple Silicon GPU — 50 epochs at 512×512 with real-time validation metrics

Model Detection

Real-time YOLO object detection identifying pantry and fridge items
